# Jukeboxd

## Description
Letterboxd for music. A mobile app that allows users to review songs and albums, similarly to Beli for restaurants and Letterboxd for movies.

Team Members: Selena Ho, Aaron Liu, Jacky Wong

## Tech Stack
- Expo (React Native framework)
- Supabase (PostgreSQL)

To view the app either
1. run locally:

   ```
   npx expo start
   ```

2. or visit this [link](https://selenaho.github.io/jukeboxd/)
